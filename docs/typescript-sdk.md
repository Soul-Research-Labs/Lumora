# TypeScript SDK Reference

> **Package**: `@lumora/sdk` | **Source**: `sdks/typescript/`

The TypeScript SDK provides a typed HTTP client for the Lumora RPC API. It wraps
all 15 endpoints with request/response types and structured error handling.

## Installation

```bash
cd sdks/typescript
npm install
npm run build
```

To use in your project (once published):

```bash
npm install @lumora/sdk
```

## Quick Start

```typescript
import { LumoraClient } from "@lumora/sdk";

const client = new LumoraClient("http://127.0.0.1:3030", "my-api-key");

// Health check (no auth needed)
const health = await client.health();
console.log(health); // "OK"

// Pool status
const status = await client.status();
console.log(
  `Balance: ${status.pool_balance}, Commitments: ${status.commitment_count}`,
);
```

## Constructor

```typescript
new LumoraClient(baseUrl: string, apiKey?: string)
```

| Parameter | Type     | Required | Description                        |
| --------- | -------- | -------- | ---------------------------------- |
| `baseUrl` | `string` | Yes      | Base URL of the Lumora RPC server  |
| `apiKey`  | `string` | No       | API key sent as `X-API-Key` header |

The client automatically:

- Strips trailing slashes from `baseUrl`.
- Prefixes all API paths with `/v1/` (except `/health`).
- Sets `Content-Type: application/json` on all requests.
- Includes `X-API-Key` header when `apiKey` is provided.

## Methods

### `health()`

Check server health. Does not require authentication.

```typescript
async health(): Promise<string>
```

**Returns**: `"OK"` if the server is running.

```typescript
const h = await client.health();
// h === "OK"
```

---

### `status()`

Get current privacy pool status.

```typescript
async status(): Promise<StatusResponse>
```

**Response type**:

```typescript
interface StatusResponse {
  pool_balance: number;
  commitment_count: number;
  merkle_root: string;
  circuit_version: string;
}
```

```typescript
const s = await client.status();
console.log(`Root: ${s.merkle_root}`);
console.log(`Circuit: ${s.circuit_version}`);
```

### `deposit(req)`

Deposit a commitment into the privacy pool.

```typescript
async deposit(req: DepositRequest): Promise<DepositReceipt>
```

**Request type**:

```typescript
interface DepositRequest {
  commitment: string; // 64-char hex string
  amount: number;
}
```

**Response type**:

```typescript
interface DepositReceipt {
  leaf_index: number;
  merkle_root: string;
}
```

```typescript
const receipt = await client.deposit({
  commitment: "aabb...64hex...",
  amount: 1000,
});
console.log(`Deposited at leaf ${receipt.leaf_index}`);
```

---

### `transfer(req)`

Submit a private transfer proof. The proof must be generated by the Rust prover.

```typescript
async transfer(req: TransferRequest): Promise<TransferReceipt>
```

**Request type**:

```typescript
interface TransferRequest {
  proof_bytes: string; // hex-encoded proof
  merkle_root: string; // current Merkle root
  nullifiers: [string, string]; // 2 input nullifiers
  output_commitments: [string, string]; // 2 output commitments
  domain_chain_id?: number; // V2 domain chain ID (optional)
  domain_app_id?: number; // V2 domain app ID (optional)
}
```

**Response type**:

```typescript
interface TransferReceipt {
  nullifiers: [string, string];
  output_commitments: [string, string];
  new_root: string;
}
```

The optional `domain_chain_id` and `domain_app_id` fields enable V2
domain-separated nullifiers for cross-chain privacy. See
[Cross-Chain Privacy](cross-chain-privacy.md) for details.

```typescript
const tx = await client.transfer({
  proof_bytes: "0a1b2c...",
  merkle_root: "abcd...",
  nullifiers: ["1111...", "2222..."],
  output_commitments: ["3333...", "4444..."],
  domain_chain_id: 1,
  domain_app_id: 0,
});
console.log(`New root: ${tx.new_root}`);
```

### `getNotes(tag)`

Fetch encrypted notes by recipient tag.

```typescript
async getNotes(tag: string): Promise<string[]>
```

Returns an array of hex-encoded ciphertexts for the given tag.

```typescript
const notes = await client.getNotes("my-tag");
for (const ct of notes) {
  // Trial decrypt with viewing key
}
```

### `history(req?)`

Query paginated event history.

```typescript
async history(req?: HistoryRequest): Promise<HistoryResponse>
```

**Request type**:

```typescript
interface HistoryRequest {
  offset?: number; // default: 0
  limit?: number; // default: 100
}
```

**Response type**:

```typescript
interface HistoryResponse {
  events: PoolEvent[];
  total: number;
}

interface PoolEvent {
  type: "Deposit" | "Transfer" | "Withdraw";
  [key: string]: unknown;
}
```

```typescript
const h = await client.history({ offset: 0, limit: 50 });
console.log(`Total events: ${h.total}`);
for (const evt of h.events) {
  console.log(`${evt.type}: ${JSON.stringify(evt)}`);
}
```

### `batchVerify(proofs)`

Batch-verify multiple transfer proofs in a single request.

```typescript
async batchVerify(proofs: TransferRequest[]): Promise<BatchVerifyResponse>
```

**Response type**:

```typescript
interface BatchVerifyResponse {
  results: boolean[]; // per-proof result
  all_valid: boolean; // true if all proofs are valid
}
```

```typescript
const result = await client.batchVerify([proof1, proof2, proof3]);
console.log(`All valid: ${result.all_valid}`);
result.results.forEach((valid, i) => {
  if (!valid) console.error(`Proof ${i} failed verification`);
});
```

### `stealthScan(req?)`

Download encrypted notes for local stealth address trial decryption. The server
returns note data without learning which notes belong to the caller.

```typescript
async stealthScan(req?: StealthScanRequest): Promise<StealthScanResponse>
```

**Request type**:

```typescript
interface StealthScanRequest {
  from_leaf_index?: number; // default: 0
  limit?: number; // default: 1000
}
```

**Response type**:

```typescript
interface StealthScanResponse {
  notes: EncryptedNoteResponse[];
  count: number;
}

interface EncryptedNoteResponse {
  leaf_index: number;
  commitment: string;
  ciphertext: string;
  ephemeral_pubkey: string;
}
```

```typescript
const scan = await client.stealthScan({ from_leaf_index: 0, limit: 500 });
console.log(`Scanning ${scan.count} notes...`);
for (const note of scan.notes) {
  // Locally: derive shared secret using view_sk and note.ephemeral_pubkey
  // If decryption succeeds, the note belongs to this wallet
  console.log(`Leaf ${note.leaf_index}: ${note.commitment}`);
}
```

See [Stealth Addresses](stealth-addresses.md) for the full scanning protocol.

## Error Handling

All methods throw `LumoraError` on non-2xx responses:

```typescript
import { LumoraClient, LumoraError } from "@lumora/sdk";

const client = new LumoraClient("http://127.0.0.1:3030", "wrong-key");

try {
  await client.status();
} catch (err) {
  if (err instanceof LumoraError) {
    console.error(`HTTP ${err.status}: ${err.body}`);
    // HTTP 401: Unauthorized
  }
}
```

**`LumoraError` properties**:

| Property  | Type     | Description             |
| --------- | -------- | ----------------------- |
| `status`  | `number` | HTTP status code        |
| `body`    | `string` | Raw response body       |
| `message` | `string` | Formatted error message |

Common error codes:

| Status | Meaning                                  |
| ------ | ---------------------------------------- |
| 400    | Invalid request (malformed JSON, etc.)   |
| 401    | Missing or invalid API key               |
| 404    | Unknown endpoint                         |
| 413    | Request body too large                   |
| 429    | Rate limited / concurrency limit reached |
| 500    | Internal server error                    |

## Complete Example

```typescript
import { LumoraClient, LumoraError } from "@lumora/sdk";

async function main() {
  const client = new LumoraClient("http://127.0.0.1:3030", "my-secret-key");

  // 1. Check health
  console.log(await client.health());

  // 2. Get pool status
  const status = await client.status();
  console.log(
    `Pool: ${status.pool_balance} balance, ${status.commitment_count} notes`,
  );

  // 3. Get fees
  const fees = await client.fees();
  console.log(
    `Fees: deposit=${fees.deposit_fee}, transfer=${fees.transfer_fee}`,
  );

  // 4. Deposit
  const receipt = await client.deposit({
    commitment: "aabbccdd".repeat(8),
    amount: 1000,
  });
  console.log(`Deposited at leaf ${receipt.leaf_index}`);

  // 5. Check nullifier
  const check = await client.checkNullifier("1122334455".repeat(6) + "1122");
  console.log(`Nullifier spent: ${check.spent}`);

  // 6. Get epoch roots
  const epochs = await client.epochRoots();
  console.log(
    `Current epoch: ${epochs.current_epoch}, ${epochs.roots.length} roots`,
  );

  // 7. Stealth scan
  const scan = await client.stealthScan({ from_leaf_index: 0, limit: 100 });
  console.log(`Found ${scan.count} notes for trial decryption`);

  // 8. History
  const history = await client.history({ offset: 0, limit: 10 });
  console.log(`${history.total} total events`);
}

main().catch(console.error);
```
